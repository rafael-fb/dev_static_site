Mel site

  1. Crie o arquivo html e garante que ele tenha as tags iniciais:
  1. Agora, dentro do head, adicione a tag de título

    <title>nome do site</title>

    A ordem das tags dentro do head não importa, pode colocar o título onde quiser.

  1. Crie uma pasta chamada css e dentro crie um arquivo chamado index.css.
  1. Faça o link entre o arquivo html e o css adicionando esta tag dentro do head:

    <link rel="stylesheet" href="./css/index.css">

  1. Por padrão todo site vem com uma margem que deixa uma ‘borda’ branca assim:

    Para remove-la, temos que ir no arquivo css e aplicar margin: 0; no body.

    body {
    margin: 0;
}
  1. Agora já podemos aproveitar e definir a cor de fundo do site que será um verde claro, basta inserir uma cor de fundo no body no valor #E4EADCFF .
  1. Vamos partir pro header:

    Primeiro, o grande bloco verde escuro:

    No html, dentro do body, adicione uma div para ele com a classe ‘header’.

    No css, estilize o header colocando:

    • cor de fundo #89998e
    • altura (height), pode ser 225px
    • largura (width) de 100% pra preencher a tela de um lado ao outro

    Isso fará um grande retangulo verde.

  1. Agora precisamos arredondar o canto inferior esquerdo do header, para isso em seu css temos que usar a propriedade border-radius. O border radius precisa de quatro valores, cada um vai arredondar um canto diferente do retangulo.

    Para arredondar apenas o canto inferior esquerdo, temos que modificar o último valor do border-radius:

    border-radius: 0 0 0 70%;

  1. Agora vamos para a imagem de perfil. Dentro do header, adicione uma div que servirá como ‘pai’ da imagem, essa div pode ter a classe ‘picture-container’.
  1. Vamos posicionar esse picture container primeiro e depois colocar a imagem dentro dele. Então no css estilize o picture container:
    • Altura de 150px
    • Largura de 150px
    • Ele terá uma borda branca:

      border: 4px white solid;

      Esse 4px é a grossura da borda, pode mudar a vontade.

    Para que ele fique redondo:

    • border radius com todos os cantos totalmente arredondados:

      border-radius: 100% 100% 100% 100%;

  1. Pronto já temos o container da foto, mas ainda falta posicionar ela no local certo… Ela vai ficar em uma posição meio estranha no header, saindo um pouco para fora dele, em casos como esse, para termos total liberdade na hora de posicionar usamos posição absoluta:
    • no .picture-container, adicione position: absolute;
  1. Para que o position absolute funcione, precisamos ir no elemento pai e adicionar position: relative; , no caso o pai do picture container é o próprio header, então no css do header, adicione position relative.
  1. Com o position absolute no picture container podemos dizer a localização exata de algo usando
    • left ou right que indica a posição na horizontal
    • top ou bottom que indica a posição na vertical

    Para o nosso caso vamos usar:

    • bottom: -10px; (valor negativo para que a imagem saia do header)
    • left: 15px;
  1. Com isso já temos o picture container no tamanho e posição corretos, agora no html, dentro do picture container adicione uma tag para a img
    1. no src dessa tag img, coloque o caminho da imagem com o nome correto, aqui no exemplo é user_pic.png, mas a sua imagem pode ter outro nome:

      <img src="./images/user_pic.png" alt="user picture">

  1. Precisamos que a imagem ocupe todo o espaço do picture container, para isso teremos que estilizar a img que está dentro do picture container: 
    .picture-container img {
		/* estilos */
}
  1. Os estilos dessa imagem serão
    • largura 100%
    • altura 100%
    • object-fit: cover; isso garante que a imagem não vai ficar distorcida
  1. Se você testar o site agora, irá notar que a imagem está vazando para fora do picture container redondo que haviamos criado, para que a imagem não vaze para fora do circulo, precisamos ir no css do .picture-container e dizer que tudo que vazar dele deve ficar escondido:

    overflow: hidden;

  1. Agora vamos para os títulos dentro do header, no html ainda dentro do header, abaixo do picture container, adicione um div com a classe header-content , essa div será o pai dos títulos.
  1. dentro dessa div header content, adicione mais 2 divs, uma para cada linha do título, adicione uma classe para cada uma dessas 2 divs: 
        <div class="header-content">
        <div class="title-1">Mel's</div>
        <div class="title-2">little place</div>
    </div>
  1. Agora vamos estilizar essas divs no css, vamos começar estilizando o header content
    • como todos os textos do header são brancos, já podemos colocar aqui no pai a cor (a cor será aplicada em todos os textos nas divs filhos)

      color: white;

    • além disso queremos que os textos fiquem centralizados

      text-align: center;

  1. Já vamos definir o tamanho das fontes dos títulos usando font-size, como cada linha tem um tamanho diferente, nós demos uma classe diferente para cada:

    title-1 e title-2

    • para o title-1 dê um font size de 50px
    • para o title-2 dê um font size de 40px
  1. Agora precisamos mover o header-content para a posição certa, para isso, no css, vamos no elemento pai do header-content que é o elemento de classe header e adicionar umas opções de estilo:
    • display flex - isso permite controlar melhor a posição dos filhos, então estamos adicionando isso no header, para controlar melhor a posição do seu filho header content
  1. Agora o header content com os títulos já pode ser movido para a posição certa usando margin:
    • margin: auto; - isso irá coloca-lo bem no meio
    • mas como bem no meio não é o ideal, podemos também aplicar margin-left de valor 45%
  1. Bem melhor, agora podemos ir para o botão, no html o botão ficará dentro do header content e abaixo do title-2, coloque a classe header-button nele 
        <div class="header-content">
        <div class="title-1">Mel's</div>
        <div class="title-2">little place</div>
        <button class="header-button">fale comigo</button>
    </div>
  1. no css, vamos estilizar o header-button:
    • para a cor de fundo do botão, coloque transparent
    • para a borda, coloque 2px cor branca e linha solida

      border: 2px white solid;

    • a cor do texto do botão pode ser branca com color: white;
    • o font size pode ser 20px
    • para arredondar as bordas usamos

      border-radius: 25px 25px 25px 25px

    • A borda está muito colada no texto do botão, precisamos engordar o botão um pouco para corrigir isso, fazemos isso com o padding
    • o padding pode receber 2 valores, o primeiro valor cresce pra cima e pra baixo e o segundo valor cresce o elemento para os lados

      padding: 5px 25px;

    • Agora só falta distanciar um pouco o botão do último título, podemos usar um margin-top com valor 15px para isso.
  1. Agora podemos ir para o desenho da estrela no canto do header, para isso, no html adicione uma div dentro do header, abaixo do header content, essa div pode ter a clsse start-container
  1. usaremos a star container para posicionar a estrela, mas a estrela em si será uma tag img dentro de star container, lembre de colocar o nome correto da imagem no src da imagem, aqui no exemplo é star.png 
    <div class="star-container">
    <img src="./images/star.png" alt="star icon">
</div>
  1. agora no css, vamos estilizar o star-container, usaremos position absolute para posiciona-lo livremente sem influenciar na posição do títulos
  1. Com o star container tendo position absolute, podemos dizer que queremos ele grudado na direita usando right: 0;
  1. O tamanho do star container deve ser altura 100% e largura de 50px
  1. Agora que já posicionas o container da estrela no local correto, vamos estilizar a imagem dentro do star container 
    .star-container img {
	/* estilos para o img dentro de .star-container */
}
  1. Os estilos dessa imagem serão:
    • width (largura) de 100%
    • altura (height) de 50%
    • object-fit: contain; - para que a imagem não se distorça
  1. Agora estamos quase lá, mas a imagem não está no meio, para que ela fique no meio, vamos precisar adicionar margin: auto;
  1. O margin auto só functiona se elemento pai for display flex, então vamo no css do .start-container e adicionar display flex
